# موقع اختبار منصف

## نشر الموقع على الإنترنت مجاناً

### الخطوة 1: إنشاء حساب GitHub
1. اذهب إلى [GitHub.com](https://github.com)
2. أنشئ حساب جديد أو سجل دخول

### الخطوة 2: رفع الكود إلى GitHub
1. أنشئ مستودع جديد (New Repository)
2. اسمه `mansaf-quiz`
3. ارفع جميع ملفات المشروع

### الخطوة 3: نشر على Vercel
1. اذهب إلى [Vercel.com](https://vercel.com)
2. سجل دخول باستخدام حساب GitHub
3. اضغط "New Project"
4. اختر مستودع `mansaf-quiz`
5. اضغط "Deploy"

### الخطوة 4: ربط قاعدة البيانات
1. في لوحة تحكم Vercel، اذهب إلى "Settings" > "Environment Variables"
2. أضف المتغيرات التالية من ملف `.env.local`:
   - `CONVEX_DEPLOYMENT`
   - `VITE_CONVEX_URL`

### معلومات مهمة:
- **كلمة مرور الإدارة**: `mansaf2024`
- **الوصول للإدارة**: اضغط على النقاط الثلاث (•••) في الزاوية اليسرى العلوية
- **الموقع سيكون متاح على**: `https://your-project-name.vercel.app`

### الحماية:
- لوحة الإدارة محمية بكلمة مرور
- فقط من يعرف كلمة المرور يمكنه رؤية النتائج
- الزوار العاديون يمكنهم فقط لعب الاختبار

## تشغيل المشروع محلياً
```bash
npm install
npm run dev
```
